01.全局与内置对象

大约 5 分钟

在javascript环境中，全局对象是代码执行时自动提供的对象，不需要显示的声明。全局对象根据运行环境(如浏览器、Node.js等)有所不同。

JavaScirpt的内置对象(也称为通用全局对象)，它是由ECMAScript标准定义的，使用于所有的JavaScript环境(如浏览器、Node.js等)，这些对象在任何标准的JavaScript环境中都可以直接访问和使用。

  1. 全局对象概述

    • 定义：全局对象是javascript环境中的顶级对象，可以直接访问其属性和方法。
    • 命名：
      • 在浏览器中，全局对象是window或globalThis.
      • 在Node.js环境中，全局对香港是global或globalThis.
      • globalThis是一个标准化的名称，跨环境通用。

  2. 浏览器中的全局对象

    • 运行时工具对象

      浏览器中，提供了 许多运行时工具对象，用于处理常见任务。

      对象说明
      console提供日志输出功能，例如 console.log()、console.error()。
      JSON提供 JSON.parse() 和 JSON.stringify() 方法，用于处理 JSON 数据。
      Math提供数学计算工具，如 Math.random()、Math.sqrt()。
      Intl用于国际化处理，例如日期、时间、货币格式化。
      Reflect提供反射工具，操作对象属性和方法，例如 Reflect.get()、Reflect.set()。
      Proxy创建代理对象，用于拦截和自定义对象行为。

    • Web API 提供的全局对象

    这些对象由浏览器提供，用于与浏览器环境交互。

    对象说明
    window浏览器的全局对象，表示当前窗口或框架。
    documentDom的入口，表示HTML文档。
    history提供浏览器历史记录的访问。
    location表示当前文档的URL信息。
    navigator提供浏览器环境的信息，例如用户代理。
    fetch提供用于网络请求的现代接口，替代XMLHttpRquest。
    XMLHttpRquest旧的网络请求接口，用于发送http请求。
    WebSocket提供实时双休通信的接口
    Event表示事件对象，用于 事件处理
    Storage提供 localStorage 和 sessionStorage，用于本地存储键值对数据。
    Worker表示 Web Worker，用于创建后台线程。

  3. Node.js中的全局对象

    • 在Node.js中，全局对象有所不同：

      对象说明
      globalNode.js的全局对象，类似浏览器中的window。
      process提供当前运行的Node.js进程的信息和控制功能，列如process.env。
      Buffer用于处理二进制数据
      setImmediate立即执行函数的工具
      require导入模块的工具
      module表示当前模块的元信息
      exports用于导出模块的接口
      __dirname表示当前模块所在的目录
      __filename表示当前模块的文件路径。

  4. 全局对象扩展

    • 自定义全局对象。

      在浏览器或 Node.js 中，可以通过直接在全局作用域中添加属性或方法的方式扩展全局对象。

      // 浏览器环境
window.myGlobalVar = "Hello, world!";
console.log(myGlobalVar); // 输出 "Hello, world!"

// Node.js 环境
global.myGlobalFunc = function () {
    console.log("This is a global function!");
};
myGlobalFunc(); // 输出 "This is a global function!"

  5. JS内置对象(通用全局对象)

    这些对象在所有的JavaScript环境中都存在，属于ECMAScript标准。

    对象说明
    Object所有对象的基类，表示通用对象类型
    Function表示函数类型，用于创建和调用函数
    Boolean表示布尔值类型
    Number表示数字类型，支持整数和浮点数
    BingInt表示大整数类型，可以表示任意大小的整数。
    String表示字符串类型，提供字符串操作方法。
    Symbol表示唯一值的基本类型，通常用于对象属性的标识符。
    Data表示日期和时间
    RegExp表示正则表达式，用于模式匹配
    Array表示数组类型
    Map键值对集合，允许任意类型的键
    Set值得集合，值不能重复
    WeakMap类似于Map,但键是弱引用
    WeakSet类似于Set,但值是弱引用
    Promise表示异步操作的结果
    Error表示错误类型的基类，用于处理错误

  6. 常见的通用全局对象用法

    1. 基本对象

      这些对象是 JavaScript 的核心，用于处理基本数据类型和通用操作。

      1. Object

        • 表示JavaScript中所有对象的基类。

          const obj = {}; // 等同于 new Object()
console.log(obj.hasOwnProperty('key')); // false

      2. Function

        • 表示函数，可以直接定义或通过构造函数创建。

          const func = new Function("a", "b", "return a + b");
console.log(func(1, 2)); // 3

      3. Boolean

        • 表示布尔值

          const bool = new Boolean(false);
console.log(bool.valueOf()); // false

      4. Symbol

        • 表示唯一标识符，通常用于定义对象的私有属性。

          const sym = Symbol("description");
console.log(sym.toString()); // Symbol(description)

    2. 数值与日期对象

      这些对象用于处理数字、日期和时间。

      1. Number

        • 用于表示和操作数值。

          const num = 123.456;
console.log(num.toFixed(2)); // 123.46

      2. BigInt

        • 用于表示任意精度的最大整数。

          const bigInt = 12345678901234567890n;
console.log(bigInt + 1n); // 12345678901234567891n

      3. Math

        • 提供数值常量和函数

          console.log(Math.sqrt(16)); // 4
console.log(Math.random()); // 生成一个 0 到 1 之间的随机数

      4. Data

        • 用于处理日期和时间

          const date = new Date();
console.log(date.toISOString()); // 当前日期的 ISO 格式

    3. 字符串和正则表达式对象

      这些对象用于处理文本和正则表达式

      1. String

        • 表示和操作字符串

          const str = "Hello, world!";
console.log(str.toUpperCase()); // HELLO, WORLD!

      2. RegExp

        • 用于定义和操作正则表达式

          const regex = /hello/i;
console.log(regex.test("Hello")); // true

    4. 集合和结构对象

      这些对象用于存储和操作数据结构。

      1. Array

        • 表示一个有序的集合

          const arr = [1, 2, 3];
console.log(arr.length); // 3

      2. Map

        • 表示键值对的集合，键可以是任意类型。

          const map = new Map();
map.set("key", "value");
console.log(map.get("key")); // "value"

      3. Set

        • 表示值得集合，不允许重复。

          const set = new Set([1, 2, 2, 3]);
console.log(set.size); // 3

      4. WeakMap

        • 类似于 Set，但只能存储对象，并对其弱引用。

          const weakSet = new WeakSet();
const obj = {};
weakSet.add(obj);

      5. WeakSet

        • 类似于 Set，但只能存储对象，并对其弱引用。

          const weakSet = new WeakSet();
const obj = {};
weakSet.add(obj);

    5. 错误对象

      用于处理和表示错误

      1. Error

        • 基本错误对象

          const error = new Error("Something went wrong");
console.log(error.message); // "Something went wrong"

    6. 异步对象

      1. Promise

        • 用于处理异步编程

          const promise = new Promise((resolve) => resolve("Done"));
promise.then(console.log); // "Done"

    7. JSON对象

      用于解析和字符串化JSON数据

      const jsonData = JSON.stringify({ key: "value" });
console.log(JSON.parse(jsonData)); // { key: "value" }
